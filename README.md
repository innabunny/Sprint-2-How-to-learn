# "Научиться учиться"

## О проекте
Типовой одностраничный сайт с информацией о различных методах обучения и ссылками на полезные ресурсы.

## Технологический стек:
### HTML
* Семантические теги
### СSS
* Flexbox
* БЭМ Nested
* Псевдоклассы CSS
* Позиционирование блоков и элементов
* Анимация и трансформация элементов
* API Youtube
